Creative Ads Manager

ยินดีต้อนรับสู่ Repository ของโปรเจค **Creative Ads Manager** ซึ่งเป็นระบบจัดการโฆษณาเชิงสร้างสรรค์แบบ Full-Stack ที่พัฒนาด้วย React, Vite, Node.js และ tRPC

ดูเอกสาร README เข้าสู่ Frontend App (ต้อง Build ก่อน)

**หมายเหตุ:** เนื่องจากโปรเจคนี้เป็นแบบ Full-Stack ส่วน Backend จะไม่สามารถทำงานบน GitHub Pages ได้ หากต้องการดูแอปพลิเคชันที่ใช้งานได้จริง กรุณา Build โปรเจคและตั้งค่า GitHub Pages ตามคำแนะนำใน README.md
